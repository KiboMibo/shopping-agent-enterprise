# Пример: поиск по фотографии

## Запрос

Пользователь загружает фотографию беспроводных наушников без
читаемой модели и просит найти такие же.

## Корректный подход

1. Зафиксировать признаки: форма кейса, ножка наушника, расположение
   индикатора, цвет, вид разъёма.
2. Не утверждать бренд только по сходству.
3. Сформировать несколько модельных гипотез.
4. Сравнить фотографии карточек с исходным изображением.
5. Исключить несовпадающие поколения и копии.
6. Для ТОП-5 указать уверенность идентификации.

## Пример вывода

> Наиболее вероятно, что на фотографии показана модель семейства X,
> но точная модификация не подтверждена: маркировка на корпусе не
> читается. В таблицу включены только визуально близкие варианты;
> перед заказом нужно сверить разъём и код на внутренней стороне
> кейса.

## Риски

- визуально похожие реплики;
- подмена поколения;
- карточка с несколькими вариантами;
- цена за один наушник вместо комплекта;
- восстановленный товар, указанный как новый.
