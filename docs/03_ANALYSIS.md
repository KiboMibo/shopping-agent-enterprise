# 03. Анализ предложений

## Единая формула надёжности

**S = 0,22A + 0,18V + 0,16R + 0,14P + 0,12D + 0,10G + 0,08C − F**

После вычисления:

`S_final = min(100; max(0; S))`

Все положительные компоненты находятся в диапазоне от 0 до 100.
Штраф `F` измеряется в баллах и вычитается после взвешивания.

Сумма положительных весов равна:

`0,22 + 0,18 + 0,16 + 0,14 + 0,12 + 0,10 + 0,08 = 1,00`

## Компоненты

### A — точность соответствия, 22%

- 90–100: точный артикул, модель и модификация;
- 70–89: вероятное совпадение с небольшими неясностями;
- 40–69: функционально близкий аналог;
- 0–39: слабое или конфликтующее соответствие.

### V — надёжность продавца, 18%

Учитываются рейтинг, число оценок, срок работы, повторяющиеся жалобы,
прозрачность реквизитов и статус официального продавца.

### R — качество отзывов, 16%

Учитываются объём, свежесть, содержательность, повторяющиеся темы,
жалобы на подделки и соответствие отзывов текущей модели.

### P — адекватность цены, 14%

Максимальный балл получает не самая низкая цена, а цена,
согласующаяся с рынком и уровнем риска.

### D — доставка и доступность, 12%

Учитываются наличие, доставка в Москву или Московскую область,
стоимость, срок, отслеживание и безопасное получение.

### G — гарантия, возврат и комплектация, 10%

Учитываются тип гарантии, срок возврата, понятность условий и
подтверждённая комплектность.

### C — подтверждённость данных, 8%

- 90–100: ключевые данные подтверждены первичными источниками;
- 70–89: подтверждено большинство полей;
- 40–69: есть существенные пробелы;
- 0–39: большая часть данных основана на заявлениях без проверки.

## Штрафы F

Штрафы суммируются:

| Риск | Штраф |
|---|---:|
| Подтверждённый признак подделки | 40–70 |
| Просьба оплатить вне площадки | 40 |
| Несовпадение модели или артикула | 20–50 |
| Подозрительно низкая цена | 10–25 |
| Жалобы на подмену товара | 10–30 |
| Серый импорт без прозрачной гарантии | 5–20 |
| Неподтверждённая комплектация | 5–15 |
| Нет информации о гарантии | 5–12 |
| Неясный регион отправки | 3–8 |
| Неподтверждённая доставка | 5–15 |

Критический риск может быть основанием исключить предложение
независимо от математического результата.

## Интерпретация

| Балл | Уровень | Интерпретация |
|---:|---|---|
| 90–100 | Очень высокий | Сильное соответствие и минимальные риски |
| 75–89 | Высокий | Хороший вариант с небольшими оговорками |
| 60–74 | Средний | Допустимо после дополнительных проверок |
| 40–59 | Низкий | Существенные риски или пробелы |
| 0–39 | Критический | Не рекомендуется |

## Анализ оригинальности

Вероятность оригинальности оценивается отдельно от `S`:

- **высокая** — официальный канал, документы, гарантия и маркировка
  согласуются;
- **средняя** — явных признаков подделки нет, но часть подтверждений
  отсутствует;
- **низкая** — есть несколько существенных красных флагов;
- **недостаточно данных** — информации мало для вывода.

## Проверка отзывов

Анализ должен отвечать на вопросы:

- какие проблемы повторяются;
- относятся ли отзывы к нужной модификации;
- насколько свежи отзывы;
- есть ли признаки стимулированных или шаблонных отзывов;
- как продавец отвечает на жалобы;
- встречаются ли жалобы на подделку или подмену.
