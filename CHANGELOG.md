# Журнал изменений

## [1.1.0] — 2026-07-10

### Добавлено

- GitHub Actions для автоматической проверки спецификации;
- локальный валидатор ссылок, формул, JSON и обязательных файлов;
- контракты поисковых и аналитических инструментов;
- уровни доказательности E0–E5;
- правила свежести динамических данных;
- обработка недоступных и противоречивых данных;
- машиночитаемый JSON-контракт ответа;
- защита от prompt injection и небезопасных платёжных инструкций;
- жёсткие ограничения поверх взвешенного скоринга;
- eval-набор для критических сценариев;
- руководство по участию и политика безопасности.

### Изменено

- служебные каталоги локальных AI-инструментов исключены из проекта;
- уточнено поведение при наличии менее пяти точных предложений.

Формат основан на принципах Keep a Changelog. Проект использует
семантическое версионирование.

## [1.0.0] — 2026-07-10

### Добавлено

- полная системная инструкция Shopping Agent Enterprise;
- компактная версия промпта;
- отдельный негативный промпт;
- модули ядра, поиска, анализа и формата ответа;
- единая формула оценки надёжности;
- шаблоны таблицы, скоринга, отзывов и рекомендации;
- примеры поиска по фотографии, артикулу, электронике,
  автозапчастям и объявлениям Avito;
- правила проверки оригинальности, цены, доставки, гарантии,
  купонов, скидок и кэшбэка;
- правила разделения новых, подержанных и восстановленных товаров;
- запрет на выдумывание или подмену неподтверждённых данных.
