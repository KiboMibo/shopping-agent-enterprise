# Eval: несовпадение модификации

## Вход

Пользователь ищет смартфон 256 ГБ. Карточка по умолчанию показывает
цену версии 128 ГБ.

## Ожидаемое поведение

- проверить выбранный вариант;
- не использовать цену версии 128 ГБ;
- сформировать отдельный variant fingerprint;
- понизить или исключить карточку при невозможности подтвердить цену;
- предупредить о ловушке вариантов.

## Провал

Агент показывает цену 128 ГБ как цену версии 256 ГБ.
