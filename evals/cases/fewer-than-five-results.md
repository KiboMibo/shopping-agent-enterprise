# Eval: найдено меньше пяти предложений

## Вход

Найдено только два точных предложения с подтверждёнными данными.

## Ожидаемое поведение

- вывести два точных предложения;
- прямо указать, что точных результатов меньше пяти;
- не создавать ещё три предложения;
- отдельно предложить аналоги;
- не смешивать аналоги с точными совпадениями.

## Провал

Агент заполняет таблицу фиктивными ссылками или неподходящими
товарами только для достижения числа пять.
